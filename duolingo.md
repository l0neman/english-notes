# 问候

## 问候 1

- 大小写

第一个词、名字、国家

```
Hello, I am John.
你好，我是约翰。
```

```
Is she from South Koren?
她来自韩国吗？
```

```
I speak Chinese.
我说中文。
```

- 提问

改变语序，使用 `to be` 提问

| 陈述句                                    | 疑问句                                      |
| ----------------------------------------- | ------------------------------------------- |
| You are from China.<br />你是从中国来的。 | Are you from China?<br />你是从中国来的吗？ |
| He is from China.<br />他是从中国来的。   | Is she from China?<br />他是从中国来的吗？  |

`where` 和 `how` 要放在问题的开头。

```
where are you from?
你来自哪里？
```

```
How is he?
他怎么样？
```

- 怎么样

问**你好吗？**或者**怎么样？**，可以说 `How are you?`

```
Hello, how are you?
I am fine. How are you?
I am fine, thanks?

你好，怎么样？
我很好，你好吗？
我很好，谢谢！
```



# 餐厅

一个、一份、一件，在英语中要用 `a`

```
I want a burger
I want a sandwich
I want a menu

我想要一个汉堡包
我想要一个三明治
我想要一份菜单
```

如果名词以 `-s` 结束，一般不用 `a`

```
I want fries.
```

`please`，**请**或**请给我**

```
Follow me, please.
A burger, please!

请跟我来
请给我一个汉堡包
```



# 家庭

- 兄弟姐妹

**姐姐**和**妹妹**都是 `sister`

**哥哥**和**弟弟**都是 `brother`

**妈妈**和**爸爸**都是 `grandparents`

- 动词

用 `to be` 这个动词代替**很**

中文**我很忙**，`to be` 放在**我**和**忙**中间，`I am busy`

```
She is busy.
He is intelligent.
You are funny!

她很忙。
他很聪明。
你很好笑！
```

# 学校

- 拥有

如果要说 `my book` 或者 `your pencil`，不要加 `a`

```
He is a Chinese teacher.
Where is your pencil?
I like my math class.

他是一位中文老师。
你的铅笔在哪里？
我喜欢我的数学课。
```

一个单词以元音开头时，前面的 `a` 要换成 `an`

（元音：元音是像 `a`、`i`、或 `u` 等的声音。发音时气流自由地经过口腔，不受阻碍）

```
I have a big family.
I have an interesting family.
I am a university professor.

我有一个很大的家庭。
我有一个有趣的家庭。
我是一位大学教授。
```

`university` 的第一个字母是 `u`，但是是辅音，所以还是要用 `a`

（辅音：辅音是像 `k`、`s`、或 `m` 等的声音。发音时气流经过口腔会受到阻碍）
